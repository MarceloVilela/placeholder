Empresas que utilizam React no Brasil

Repositório responsável em listar empresas e projetos que utilizam nosso querido React.

Versão Mobile · Versão Desktop


Também confira nossos outros projetos
Eventos sobre React · Procurando talentos? · Procurando emprego?

Introdução

Se sua empresa e/ou projeto utiliza alguma tecnologia do ecossistema React, não deixe de adicionar aqui nesta lista, abrindo um Pull Request.

Alguns projetos são de startups ainda em concepção ou incubadas dentro de empresas que podem ter potencial de virar produto independente e merecem destaque, indique o seu projeto caso se enquadre nessas situações.

Caso tenha dificuldades sempre pode perguntar no Slack.

Contribuições, dúvidas e melhorias

Para propor melhorias, basta abrir uma issue nesse repositório. Mais fique atento se a issue já não foi feita antes e antes de realizar sua contribuição dê uma olhada nas regras e desde já agradecemos.
